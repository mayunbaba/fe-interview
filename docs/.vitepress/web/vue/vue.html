Vue 数据双向绑定原理是通过 数据劫持 + 发布者-订阅者模式 的方式来实现的。

vue双向绑定实现 ​

  1. observer 劫持数据
  • 递归代理数据，或者递归劫持数据属性。
  • 数据变化通知订阅者
  1. compile 解析模板指令
  • 把模板中的变量解析成数据
  • 根据指令绑定事件
  • 添加订阅者，如果数据变化，收到通知，更新视图。
  1. Watcher类是连接observer和compile的桥梁。扮演事件总线的角色
